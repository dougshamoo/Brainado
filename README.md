## Usage

> Generates visualization of word associations using d3 and React to enhance idea generation

## Requirements

- Node.js 5.2.0
- Express.js ^4.13.3
- Material-UI ^0.14.3
- React ^0.14.6
- ReactDOM ^0.14.6
- Superagent ^1.6.1

## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

This project is deployed on Heroku.
